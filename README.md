## 🎯 서비스 소개

**학업 운세 슬롯머신**은 대학생들의 학업 생활에 대한 재미있는 운세를 제공하는 웹 애플리케이션입니다.

- 🎰 **3개의 슬롯 릴**: 학업 카테고리, 상태, 결과 조합
- 🤖 **AI 기반 운세 생성**: Google Gemini 2.0 Flash 모델 활용
- 💬 **공감형 콘텐츠**: 대학생 특화 유머러스하고 위로되는 메시지

## ✨ 주요 기능

- **실시간 AI 운세 생성**: Gemini API를 통한 창의적이고 맥락에 맞는 운세 해석
- **부드러운 슬롯 애니메이션**: 카지노 슬롯머신 스타일의 릴 회전 효과

## 🚀 로컬 실행 방법

### 1. 사전 요구사항

- **Node.js** (v16 이상 권장)
- **npm** 또는 **yarn**
- **Google Gemini API 키** (발급 방법은 아래 참조)

### 2. 저장소 클론 및 패키지 설치

```bash
git clone <repository-url>
cd slot-machine
npm install
```

### 3. 환경변수 설정 (`.env` 파일 생성)

프로젝트 루트에 `.env` 파일을 생성하고 다음과 같이 작성:

```env
GEMINI_API_KEY=your_gemini_api_key_here
PORT=3000
```

- `GEMINI_API_KEY`: Google AI Studio에서 발급받은 API 키
- `PORT`: 서버 포트 (기본값: 3000)

### 4. 서버 실행

```bash
npm start
```

또는

```bash
node server.js
```

### 5. 브라우저 접속

```
http://localhost:3000
```

## 🔑 API 키 발급 방법

### Google Gemini API 키 발급

1. **Google AI Studio 접속**  
   👉 [https://aistudio.google.com/](https://aistudio.google.com/)

2. **로그인**  
   구글 계정으로 로그인

3. **API Key 생성**

   - 좌측 메뉴에서 `Get API key` 클릭
   - `Create API key` 버튼 클릭
   - 생성된 API 키 복사

4. **`.env` 파일에 추가**
   ```env
   GEMINI_API_KEY=AIza...your_key_here
   ```

1. **메인 화면**: 3개의 슬롯 릴과 중앙의 버튼이 보입니다.
   ![alt text](images/initial.png)

     ![alt text](images/answer.png)

   - 뽑힌 조합 (예: `보고서 × 🤔생각많은 × 🐢거북이`)

## ⚙️ API 제한사항

### Gemini 2.0 Flash 모델 무료 티어 제한

| 항목                          | 제한      |
| ----------------------------- | --------- |
| **RPM** (Requests Per Minute) | 15        |
| **TPM** (Tokens Per Minute)   | 1,000,000 |
| **RPD** (Requests Per Day)    | 200       |

- **일일 최대 200회 운세 생성 가능**
- 분당 15회 이상 요청 시 일시적으로 제한될 수 있음
